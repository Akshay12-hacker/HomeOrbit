# Step 1 — Create project fresh (avoids ALL version conflicts)

```
npx create-expo-app@latest HomeOrbit --template blank
```
```
cd HomeOrbit
```

# Step 2 — Install navigation packages

```
npx expo install @react-navigation/native @react-navigation/native-stack @react-navigation/bottom-tabs react-native-screens react-native-safe-area-context expo-linear-gradient
```

# Step 3 — Copy your source files into src/
> (paste the screens/, components/, services/, hooks/, navigation/, theme/ folders)

# Step 4 — Replace App.js with the provided one

# Step 5 — Run

```
npx expo start
```
